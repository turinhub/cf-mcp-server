# CF-MCP服务器产品背景

## mcp-image-generation
- 解决需求：为AI应用提供标准化的图像生成接口
- 使用场景：
  - 内容创作工具集成
  - 营销素材自动生成
  - 产品原型可视化

## mcp-jina-api
- 解决需求：统一的多模态搜索服务
- 使用场景：
  - 文档语义搜索
  - 跨媒体内容检索
  - 知识库增强

## mcp-tavily-api  
- 解决需求：专业领域搜索服务
- 使用场景：
  - 研究数据收集
  - 市场情报分析
  - 实时信息获取

## 用户体验目标
1. 统一的API设计规范
2. 一致的错误处理机制
3. 可扩展的认证集成
4. 详细的开发者文档
